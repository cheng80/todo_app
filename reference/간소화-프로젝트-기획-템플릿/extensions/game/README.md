# 게임 확장 사용법

이 폴더는 게임 프로젝트일 때만 사용한다. 일반 앱 프로젝트에는 복사하지 않는다.

## 추가 방법

1. `게임-기획-확장.md`를 프로젝트의 `docs/product/`로 복사한다.
2. 기본 `기획서.md`에는 게임의 MVP와 제외 범위를 적는다.
3. 게임 전용 문서에는 핵심 루프, 상태, 규칙, 콘텐츠, 밸런스만 적는다.
4. 기능별 구현 계획은 기존 `docs/plans/TASK-001-기능명.md` 형식을 계속 사용한다.

게임 시스템이 작다면 모든 내용을 한 문서에 적는다. 별도 레벨·경제·밸런스 문서는 실제로 관리 부담이 생길 때만 나중에 추가한다.
